Contrato do Projeto da Unidade Curricular de Robótica 2017/2018

Aluna: Ângela Filipa Pereira Cardoso

Professor: Armando Jorge Miranda de Sousa

1. Como projeto final da unidade curricular de Robótica, a aluna compromete-se a usar a aplicação Robobo Developer, que permite programar o robô Robobo usando ROS, para o fazer seguir linhas de uma cor a escolher pelo utilizador.

2. A aluna terá acesso ao Robobo, incluindo a base robótica e um telemóvel Android, à aplicação Robobo Developer e à sua documentação.

3. A aplicação ROS a desenvolver deve usar a câmara frontal do telemóvel como único sensor para seguir linhas coloridas em papel branco.

4. À falta de um simulador Gazebo do Robobo, numa primeira fase, a aluna utilizará o simulador Turtlebot3 Waffle, que em forma se aproxima mais ao Robobo.

	4.1. A aluna deve garantir que não é utiliza a funcionalidade de profundidade da câmara do Turtlebot3 Waffle, uma vez que esta funcionalidade não está disponível no Robobo.

5. Na entrega intermédia do projeto (Assignment 3), a 14-12-2017, a aluna compromete-se a ter o simulador Turtlebot3 Waffle a seguir linhas pretas em fundo branco.
	
	5.1. Inicialmente o robô deverá estar posicionado sobre a linha preta.

	5.2. Alternativamente o robô deve andar aleatoriamente até encontrar uma linha preta para seguir.

6. Na entrega final do projeto (Assignment 4), a 05-01-2017, a aluna deve ter o Robobo a seguir linhas de uma de três cores (vermelho, verde e azul) a escolher pelo utilizador, em papel branco. A aplicação desenvolvida deve permitir calibrar a Robobo para detetar devidamente a cor seguir.

7. Para crédito extra, a aluna poderá adaptar a aplicação Robobo Developer de forma a que fique disponível para utilizar a câmara traseira do telemóvel.

	7.1. Deve ser disponibilizado à aluna o código fonte da aplicação Robobo Developer e, se possível, a respetiva documentação.

8. Alternativamente, caso não seja disponibilizado o código fonte da aplicação Robobo Developer, a aluna poderá realizar, para crédito extra, uma implementação para Robobo da linguagem OzoCodes - https://ozobot.com/play/color-code-language.
